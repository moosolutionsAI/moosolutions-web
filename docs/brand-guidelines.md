# MooSolutions.ai — Brand Guidelines

## Brand Identity

**Name**: MooSolutions.ai
**Tagline**: "Less interface. More you."
**Manifesto**: "We build tools that disappear."

MooSolutions.ai is the parent brand and product studio behind AI-powered tools that remove friction from how people work. Every product exists because a moment of frustration demanded a better answer.

## Brand Voice

### Tone Attributes
1. **Quietly confident** — never shouts, never needs to
2. **Precise and economical** — earns every word, wastes none
3. **Warmly intelligent** — smart without being cold, friendly without being chatty
4. **Understated wonder** — treats genuine innovation as a natural fact, not a sales pitch

## Color Palette

### Primary

| Name | Hex | Usage |
|------|-----|-------|
| Void Black | `#09090B` | Primary background |
| Surface Dark | `#141417` | Cards, section backgrounds |
| Elevated Surface | `#1E1E24` | Hover states, overlays |

### Accent

| Name | Hex | Usage |
|------|-----|-------|
| Deep Violet | `#7C3AED` | Primary accent, CTAs, energy |
| Soft Violet | `#A78BFA` | Glows, highlights, halos |
| Violet Shadow | `#2A1F3D` | Gradient fills, depth layers |

### Text

| Name | Hex | Usage |
|------|-----|-------|
| Luminance White | `#F0F0FF` | Hero headlines, key callouts |
| Ghost White | `#E4E4F0` | Primary body text |
| Muted Steel | `#6B6B7A` | Secondary text, captions |

### Rules
- Never use pure white (#FFFFFF) — always warm it slightly
- Violet accent at full saturation only on interactive elements and key moments
- Use Violet Shadow for subtle gradient fills, never as text color
- Noise grain texture at 3-5% opacity on all dark surfaces

## Typography

### Font Stack

| Role | Font | Weight | Tracking |
|------|------|--------|----------|
| Display / Hero | Cabinet Grotesk or Neue Montreal | Bold/Black | Wide (0.02em) |
| Technical Labels | Geist Mono | Regular | 0.2em |
| Body | Inter Variable | Regular/Medium | Normal |
| Accent/Eyebrow | Geist Mono | Regular | 0.2em, in Deep Violet |

### Scale

| Level | Size | Usage |
|-------|------|-------|
| Hero | 96-120px | Main hero headline only |
| H1 | 64-72px | Section headlines |
| H2 | 36-48px | Sub-headlines |
| H3 | 24-28px | Card titles |
| Body | 16-18px | Paragraph text |
| Caption | 13-14px | Labels, meta text |
| Mono Label | 12-13px | Eyebrow labels, tech tags |

### Rules
- Aggressive scale contrast — nothing between hero and body should feel "medium"
- Display type is always uppercase or title case, never sentence case
- Body text line-height: 1.6-1.7
- Headline line-height: 1.0-1.1

---

## Visual Elements

- Fractured light refractions / chromatic aberration on hover states
- Noise grain texture at 3-5% opacity on all dark surfaces
- Violet radial glow behind product visuals and hero elements
- Thin 1px borders in `rgba(124, 58, 237, 0.15)` for card definition
- No heavy drop shadows — depth through layered surfaces and glow
- No rounded corners on primary CTAs (brutalist-refined)
